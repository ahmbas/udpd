# udpd
Simple AF UDP Proxy Server

```
NAME:
   udpd - Simple AF UDP proxy server

USAGE:
   udpd [global options] command [command options] [arguments...]

VERSION:
   1.1.1

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --src_host value  Host to bind (default: "127.0.0.1")
   --src_port value  Port to listen to (default: 12345)
   --dst_host value  Host to forward to (default: "127.0.0.1")
   --dst_port value  Port to forward to (default: 54321)
   --help, -h        show help
   --version, -v     print the version
```
